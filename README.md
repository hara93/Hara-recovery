# HARA Recovery

Next.js PWA aplikacija za rehab kuka/SI, trening snage i povratak nogometu.

## Deploy na Vercel
1. Uploadaj SVE datoteke i mape iz ovog foldera u GitHub repo root.
2. Vercel -> Project -> Redeploy.
3. Otvori link i na iPhoneu: Share -> Add to Home Screen.

## Lokalno pokretanje
npm install
npm run dev
